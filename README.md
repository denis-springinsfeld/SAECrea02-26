# Woocommerce Product Importer

Télécharger : wordpress.org/plugins/woocommerce/

Ce dossier contien un dossier `sample` qui contient un fichier `products.csv` qui est un exemple de fichier CSV pour l'importation de produits dans WooCommerce. Vous pouvez utiliser ce fichier comme modèle pour créer votre propre fichier CSV avec les données de vos produits.
